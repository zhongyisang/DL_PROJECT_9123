ABSA

Aspect Based Sentiment Analysis, PyTorch Implementations.
基于方面的情感分析，使用PyTorch实现。

主讲老师: 罗老师

￼

Requirement

  • torch>=1.6.0
  • numpy >= 1.13.3
  • sklearn>=0.0
  • python 3.6

To install requirements, run pip install -r requirements.txt.

Usage

Training

python train.py --model_name memnet --dataset restaurant
